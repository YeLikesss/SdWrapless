# SdWrapless
&emsp;&emsp;适用于DMM/Fanza Galgame软电池DRM提取

## 环境
&emsp;&emsp;.Net 6.x

## 使用方法
&emsp;&emsp;1. 拖拽`软电池程序`到窗口

&emsp;&emsp;2. 点击`提取`按钮

&emsp;&emsp;3. 提取后文件放在游戏路径下`SdWrap_Extract`文件夹中

## 已通过测试版本与加密类型(其他的没找到测试源)
&emsp;&emsp;SdWrapV4+

&emsp;&emsp;软电池与Exe打包(不加密/内存加密)

&emsp;&emsp;软电池与Exe分离(不加密)
